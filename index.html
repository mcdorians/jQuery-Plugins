jQuery - Wheelclock

Time Mode

How to use:
  • import jquery
  • import plugin
  • import css and change it as you like to adjust the layout to your needs
                <link href="jquery.wheelclock.css" type="text/css" rel="stylesheet" />
                <script type="text/javascript" src="jquery.wheelclock.min.js"></script>
                <script>
                    $('#normal').wheelclock();
                </script>

Stopwatch

toggle - refresh
How to use:
  • pass the 'stopwatch'
  • create buttons
  • let the buttons trigger 'toggle' and 'reset'
                $('#stop').wheelclock({
                    mode:'stopwatch'
                });
                //Buttons made simple with jQuery 
                $('#toggle').click(function(){
                    $('#stop').wheelclock('toggle');
                });
                $('#refresh').click(function(){
                    $('#stop').wheelclock('reset');
                });

More options

  • mode: can be either 'clock' or 'stopwatch', default: 'clock
  • running: if true then the clock is ticking, on false its stoped, default: true
  • seconds: if true then the seconds count is displayed, default: true
  • speed: value from 0 to 900, specifys the time for the rotation, default : 500
                $('#options').wheelclock({
                    mode:'clock',
                    running:'true',
                    seconds: false,
                    speed: 200
                });

More methods

  • toggle: toggles between running and not running
  • reset: rests the clock depending on the mode. In clock-mode it return to the current system time an in stopwatch-mode it return to zero.
  • destroy: unbinds the timer and clears the container
                $('#box').wheelclock('toggle');
                $('#box').wheelclock('reset');
                $('#box').wheelclock('destroy');
